Description

one more 以旅選文、旅選物、旅選色、旅選輯的四大分類介紹日本，一篇深入的旅遊文化探討 ，也可能是一個值得購買的紀念品開箱，一張曾經按下的美好瞬間，或是日本當地進行的特別企劃。整體是以資訊內容為主，網站設計則以簡潔、典雅的風格走向，讓 user 在閱讀的時候能夠舒適的瀏覽。
